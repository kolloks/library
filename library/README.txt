Библиотека. Основной сценарий: человек может брать одну или несколько книг в библиотеке. Одна и таже книга может быть у разных людей в разное время.
БД - H2 - заполняется при старте информацией из initial-data.yml.
Приложение позволяет:
-Редактировать справочник книг
-Редактировать справочник людей, которые берут книги
-Брать и сдавать книгу в библиотеки
-Просматривать, какие книги в данные момент у человека
-Просматривать, кто из людей брал определенную книгу
Рассмотрены 3 уровня доступа: 
- Админ (Admin.java): /admin
- Пользователь (Library.java): /take, /return
- Гость (Application.java): /
